1- en la consola de VS Code: firebase login    
(luego presionar enter)(asi nos debe quedar: 
PS C:\Users\Edgardo\Desktop\sistema> firebase login
Already logged in as info@europaenvios.com)

2- una vez cargado debemos ingresar: firebase deploy --only hosting 
(luego presionar enter)

3- si nos aparece este error: 
Error: Authentication Error: Your credentials are no longer valid. Please run firebase login --reauth
debemos restarurar firbase asi: firebase login --reauth 
(luego presionar enter)
aqui nos debemos logear con info@eruopaenvios.com en la pagina que nos abre y dar permisos.


4- una vez restaurada la conexion con firebase debemos seguir con este comando:
firebase deploy --only hosting
(Luego precionar enter)

5- con el paso 4 ya podemos ingresar al link que nos da.
